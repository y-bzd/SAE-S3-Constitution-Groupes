PROJET S301 - BASE DE DONNÉES - BUT 2


DESCRIPTION :
Implémentation de la base de données de gestion des affectations (SAE S3.01).
Compatible MySQL / MariaDB.

CONTENU DU LIVRABLE :
1. creation_tables.sql     : Nettoyage (DROP) et Création (CREATE) des tables.
2. insertions.sql  : Jeu de données réaliste (>20 lignes).
3. Vues.sql : Vues pour l'application (Stats).
4. Triggers.sql : Triggers de gestion (Capacité, Notes, Historique, Algo).
5. Fonctions.sql     : Fonctions de calcul (Moyenne) et procédures d'inscription.

INSTRUCTIONS D'INSTALLATION :
Veuillez exécuter les scripts DANS L'ORDRE STRICT (de 1 à 5) pour garantir 
le respect des clés étrangères et des dépendances.
